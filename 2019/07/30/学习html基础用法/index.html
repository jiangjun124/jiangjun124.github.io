学习html基础用法

2019-07-30

html

一、概念

  • 简单介绍
    html 又叫超级文本标记语言，其中包括字符，图片，超链接，视频，音频，载体是通过文档来实现的.

  • 架构
    建立于B/S架构 B为浏览器/S为服务器

  • 网络概念
    Http 超文本传输协议
    url 统一资源定位符
    uri 统一资源标识符

    二、html网页编写

  • 开发流程

    1. 编写网页(我用的是sublime和vscode)
    2. 本地测试
    3. 部署(linux)将本地的html网页上传至云服务器，并且部署到apache静态服务器上.
    4. 网络测试

  • 网页元素

    • 注释
      格式:<!--注释内容-->
      <option value=""></option>

    • 属性

      1. 核心属性

        1. 1.id 文档内部唯一标识
          2.class 类,用法

          3.title 描述
          4.style 样式，其取值为css规则，css规则：键值对，键值之间通过冒号分割，规则之间通过分号分割，eg：
          <div id='one' style="color:#333;backgroud:#ededed"></div>

      2. 特有属性

        1. 1.a 其中有href和target.
          2.img 格式为alt和src

      3. 拓展属性

        1. data-xxx,eg:
          <div data-nn='one'></div>

    • 结构

      1. doctype 声明部分.
      2. html
        1.head  不会显示到浏览器视口中,head中有:
        title    显示在浏览器选项卡
        meta    用于设置元素信息配置
        charset    utf-8/gbk
        编码    编辑器UTF-8
        解码    浏览器
        2.body  网页主体

    • 元素

      1. 块元素 (box)
        作用: 作为页面框架,或者容器。是页面的主题
        特性:
        1) 独占一行空间
        2）默认宽度为100%，默认高度由子元素或者内容决定
        3) 可以为其设置指定宽度，用style=”width:height;”
      2. 行内元素
      3. 功能元素
        • 表格
        • 表单
      4. 子元素
