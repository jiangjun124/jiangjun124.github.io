学习html基础用法

2019-07-30

html

一、概念

  • 简单介绍
    html 又叫超级文本标记语言，其中包括字符，图片，超链接，视频，音频，载体是通过文档来实现的.

  • 架构
    建立于B/S架构 B为浏览器/S为服务器

  • 网络概念
    Http 超文本传输协议
    url 统一资源定位符
    uri 统一资源标识符

    二、html网页编写

  • 开发流程

    1. 编写网页(我用的是sublime和vscode)
    2. 本地测试
    3. 部署(linux)将本地的html网页上传至云服务器，并且部署到apache静态服务器上.
    4. 网络测试

  • 网页元素

    • 注释
      格式:<!--注释内容-->
      <option value=""></option>

    • 属性

      1. 核心属性

        1. 1.id 文档内部唯一标识

          2.class 类,用法

          3.title 描述

          4.style 样式，其取值为css规则，css规则：键值对，键值之间通过冒号分割，规则之间通过分号分割，eg：
          <div id='one' style="color:#333;backgroud:#ededed"></div>

      2. 特有属性

        1. 1.a 其中有href和target.
          2.img 格式为alt和src

      3. 拓展属性

        1. data-xxx,eg:
          <div data-nn='one'></div>

    • 结构

      1. doctype 声明部分.
      2. html
        1.head 不会显示到浏览器视口中,head中有:
        内容 作用
        title 显示在浏览器选项卡
        meta 用于设置元素信息配置
        charset utf-8/gbk
        编码 编辑器UTF-8
        解码 浏览器
        2.body 网页主体

    • 元素

      1. 块元素 (box)
        作用: 作为页面框架,或者容器。是页面的主题
        特性:
        1) 独占一行空间

        2）默认宽度为100%，默认高度由子元素或者内容决定

        3) 可以为其设置指定宽度，用style=”width:height;”

        块元素 意义
        div 无意义的块元素
        h1~h3 标题
        p 段落
        ul、li 无序列表、列表项
        ol、li 有序列表、列表项
        dl、dt、dd 有序列表

        H5新增的块元素(语义化标签)
            header(头)
            nav
            article(主体)
            section(部分)
            footer(脚)
            aside

      2. 行内元素
        作用: 点缀网页，填充内容
        特性:
          1)与其他行内元素共享一行空间

          2)默认宽高由内容决定

          3)不能为其指定宽和高

          4)行内元素不可以嵌套块元素，反之可以

          span是无意义的行内元素

          a是超级链接(类似于axure open link)

          默认样式有color,text-decoration,cursor.默认事件是(js_dom)点击事件

          img src为图片地址，格式为base64的数据，alt是图片找不到的时候文本替换
          strong、bold、b、em、i、sub、sup、del…

      3. 功能元素

        • 表格(table)
        表格组成 属性或者意义
        表格属性 border(边框)、width(宽度)…
        caption 表格标题
        thead 表头部分
        tbody 表格主体部分(不可省略)
        tfoot 表底部分

        • 表单(前后台交互) 用户->表单->后台->数据库

          1. form语法为：<form action="./" mtthod="post/get"></form>
            属性 功能或者用法
            action 后台接口地址
            method 请求方式get：参数拼接在url后面，通过？来分割，传递参数较少，post：参数存放保存在请求体中，安全，传递参
            enctype 编码方式
            ajax 查询字符串
          2. input
            属性名 功能
            name 不能省略作为参数中的key
            value 作为参数中的value,在按钮中务必指定value值
            type 类型
            type包含的类型
            名字 作用
            text 单行文本框
            password 密码框
            submit 提交按钮
            file 附件选择器
            radio 单选按钮
            checkbox 复选按钮
            checked 单值属性，默认地址
            placeho 提示语
          3. select
            option:标签中的文本显示在网页中，提交的值应该是option的value值，当这个值没有设定的时候，提交的是标签中的文本
          4. textarea
            多行文本域

      4. 子元素
